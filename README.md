# TuPrimeraPaginaSalamida

Proyecto web en Django estilo blog.

# Funcionalidades
- Crear Autor
- Crear Categoría
- Crear Post
- Buscar Post por título

# Cómo probar
1. Ir a /
2. Crear autor en /autor/
3. Crear categoría en /categoria/
4. Crear post en /post/
5. Buscar en /buscar/

# Tecnologías
- Python
- Django
- SQLite
